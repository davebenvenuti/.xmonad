My xmonad config

Setup
=====

```
git clone git@github.com:davebenvenuti/.xmonad.git ~/.xmonad
cd ~/.xmonad
./bootstrap
```

Keybindings
===========

Switch Workspaces: CTRL-ALT-[left arrow/right arrow]

Lock Screen: CTRL-L

Everything else is xmonad standard, with the exception that alt1 is mapped to super
